### **_Description_**

## Why pay a subscription to post beats online when I can just build my own website? This website will give users access to music catalouge and information for bookings and/or appointments.

### **_Technologies Used_**

- HTML
- JavaScript
- CSS
- Mongoose
- Express
- React
- Node
